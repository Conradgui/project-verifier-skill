# Task 0 Brief: Pre-Execution Plan Review

## Objective

Independently review the approved V3 design, implementation plan, and multi-agent orchestration contract before Task 1 begins.

## Scope

- Read-only review.
- Check dependency ordering and shared-file ownership.
- Check TDD RED/GREEN, CI, and commit timing.
- Check command executability and historical-test migration.
- Check subagent and inline fallback promises.
- Check deletion and rename timing.

## Forbidden Actions

- Do not modify files.
- Do not install dependencies.
- Do not use network access.
- Do not commit, push, merge, or update the installed Skill.

## Required Report

- `status: APPROVED | CHANGES_REQUIRED`
- Critical, Important, and Minor findings with file and line evidence.
- Minimal corrections.
- Explicit verdict on whether Task 1 may start.
